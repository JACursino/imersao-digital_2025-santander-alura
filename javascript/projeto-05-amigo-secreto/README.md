# Projeto Ingresso Online

Este projeto é uma simulação de um sistema de compra de ingressos online, desenvolvido como um desafio para aprimorar habilidades em lógica de programação.

## Funcionalidades

*   Permite ao usuário selecionar o tipo de ingresso (Pista, Cadeira Superior, Cadeira Inferior) e a quantidade desejada.
*   Exibe a quantidade de ingressos disponíveis para cada tipo.
*   Impede a compra de ingressos caso a quantidade solicitada seja maior do que a disponível, exibindo um alerta ao usuário.

## Tecnologias Utilizadas

*   HTML
*   CSS
*   JavaScript
